Bank Loan Classification Models

OVERVIEW :-

In this project, we applied several classification models to predict whether a person is eligible for a bank loan. The models used include:

  • Logistic Regression
  • Decision Tree Classifier
  • Random Forest Classifier


COLUMN DESCRIPTION

Age: The age of the applicant.

Experience: Number of years of professional experience.

Income: Annual income of the applicant.

Family: Size of the applicant's family.

CCAvg: Average spending on credit cards per month.

Education: Level of education (e.g., undergraduate, graduate, professional).

Mortgage: Value of the mortgage if any.

Personal Loan: Whether the applicant has taken a personal loan before (Yes/No).

Security Account: Whether the applicant has a security account (Yes/No).

CD Account: Whether the applicant has a certificate of deposit (CD) account (Yes/No).

Online Apply: Whether the applicant applied online for the loan (Yes/No).

Based on these features, we predict whether the person is eligible to get the loan using the models mentioned above.
